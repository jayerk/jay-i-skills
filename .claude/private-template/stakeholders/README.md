# Stakeholders Directory

## Purpose

This directory contains stakeholder maps and influence strategies for active initiatives. Each file maps the key people involved in an initiative — their roles, positions, motivations, and the engagement strategy for each.

## How to Use

1. Create one file per initiative: `[initiative-name].md`
2. Use the stakeholder map format from `skills/stakeholder-navigation/SKILL.md`
3. Update positions and strategies as they change — stakeholder maps are living documents

## Template

```markdown
# Stakeholder Map: [Initiative Name]

## Decision Context
[What decision or outcome we're working toward]

## Key Players

| Stakeholder | Title/Role | Authority | Influence | Position | Key Concern |
|-------------|-----------|-----------|-----------|----------|-------------|
| [Name] | [Title] | [H/M/L] | [H/M/L] | [Champion/Supportive/Neutral/Skeptical/Blocker] | [What they care about] |

## Engagement Strategy

### [Stakeholder Name]
- **Role in decision:** [Sponsor/Approver/Influencer/Gatekeeper]
- **What they need to hear:** [Framed in their priorities]
- **Best format:** [1:1/Demo/Memo/Data]
- **When to engage:** [Timing relative to decision]
- **Who should engage them:** [You/Sponsor/Peer]
- **Status:** [Not yet engaged / In progress / Aligned]

### [Next Stakeholder]
...

## Engagement Sequence
1. [First action] — [Stakeholder] — [Timeline]
2. [Second action] — [Stakeholder] — [Timeline]
3. [Decision point] — [Forum] — [Timeline]

## Risk Watch
- [Risk to alignment] — [Trigger to watch for] — [Response if triggered]
```

## Tips

- Be honest about positions — a map that shows everyone as "supportive" isn't useful
- Update after every significant interaction
- Track movement — is a skeptic becoming neutral? Is a champion losing interest?
- Note the informal influencers, not just the formal authority holders

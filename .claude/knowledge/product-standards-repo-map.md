# Product Standards — Sub-Practice to Repo Map

Quick reference for coaches and assessors: when a team scores low on a
sub-practice, load these files to guide the improvement conversation.

Use after completing a scorecard (`templates/SCORECARD-TEMPLATE.md`). The
Priority Action column in the scorecard should point to the starting framework
phase listed here.

---

## How to Use This Map

1. **Score the team** using the scorecard template
2. **Find the sub-practice** with the largest gap in the lowest adoption wave
3. **Load the files** listed in that row — framework first, then skill, then knowledge
4. **Start at the framework phase** noted in the coaching guidance column

## Secondary Skill Cross-References

Some sub-practices benefit from a second skill when the gap involves a specific
work mode beyond coaching or strategy.

| Sub-Practice | Secondary Skill | When to Load |

| CMI, S&P, CEJ | `skills/work/prd-writing/SKILL.md` | When research findings need to flow into PRDs — discovery-to-documentation handoff |

| MO, S&R | `skills/work/stakeholder-navigation/SKILL.md` | When strategy alignment requires stakeholder mapping or engagement sequencing |
| ADP, RfA | `skills/work/internal-tooling/SKILL.md` | When the team needs a dashboard, scorecard, or tracking tool to operationalize the routine |
